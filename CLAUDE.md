# B2B Tennis — 테니스 동호회 복식 리그 기록 사이트

## 프로젝트 개요
테니스 동호회 복식 게임 리그 기록 웹사이트.
- **스택:** Next.js 14 App Router + Supabase (PostgreSQL + Auth) + Tailwind CSS
- **점수:** 승리 3점 + 세트/게임 득실차 포함 복합 점수 순위

## 하네스: Tennis Dev

**목표:** 에이전트 팀이 B2B Tennis 사이트를 설계·구현·검증한다.

**트리거:** 사이트 개발, 기능 구현, 페이지 추가, 버그 수정 등 개발 작업 요청 시 `tennis-orchestrator` 스킬을 사용하라. 단순 질문이나 코드 설명은 직접 응답 가능.
